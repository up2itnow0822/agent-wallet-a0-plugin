# Changelog

## [1.0.0] - 2026-03-23

### Added
- Initial release
- Non-custodial crypto wallet plugin for Agent Zero
- Check ETH and ERC-20 token balances on Base, Ethereum, Arbitrum, Polygon
- Send ETH and ERC-20 tokens with configurable spend limits
- Token swaps via Uniswap V3 with slippage and fee tier configuration
- Cross-chain USDC bridging via CCTP (Circle)
- x402 HTTP payment protocol support
- Per-transaction and daily USD spend limits for agent safety
